# 💰 Expense Tracker Pro

A modern Expense Tracker web application built using **Python, Flask, SQLite, HTML, CSS, and JavaScript**.

---

## 📌 Features

- ✅ Add Expense
- ✅ Edit Expense
- ✅ Delete Expense
- ✅ Search Expenses
- ✅ Filter by Category
- ✅ Expense Analytics (Pie Chart)
- ✅ Monthly Budget Management
- ✅ Remaining Budget Calculation
- ✅ Export Expenses as PDF
- ✅ Responsive Dashboard

---

## 🛠️ Technologies Used

- Python
- Flask

## 🚀 Installation

```bash
git clone https://github.com/Apsana20/Expense-Trackor.git

cd Expense-Trackor

pip install -r requirements.txt

python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
